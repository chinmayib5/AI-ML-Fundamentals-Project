This project demonstrates the application of Bayes’ Theorem using a medical testing scenario. It explains how conditional probability helps in decision-making under uncertainty.

### Key Concepts Covered
- Bayes’ Theorem Formula
- Conditional Probability
- Medical Diagnosis Example
- False Positive & True Positive Rates
- Application in Machine Learning
